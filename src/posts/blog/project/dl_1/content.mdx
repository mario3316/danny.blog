## 프로젝트 개요

Valve 사의 신작 게임인 `Deadlock`을 플레이하면서, 인게임의 일부 기능을 게임 외부에서도 사용하고 싶다는 생각이 들어 개발을 시작한 프로젝트입니다.
인게임 내에서 아이템 정보를 확인하고 빌드를 만들어볼 수 있는데, 해당 기능부터 구현을 시작으로 추가적으로 필요하다 싶은 기능이 있으면 계속해서 추가해나갈 계획입니다.

현재 개발 계획중인 기능은 아래와 같습니다.

-   아이템 정보 조회
-   캐릭터 정보 조회
-   아이템 빌드 생성
-   아이템 빌드 비교

![실제 인게임 아이템 정보 조회 기능](/blog/posts/project/dl_1/imgs/1.webp)
![실제 인게임 아이템 빌드 생성 기능](/blog/posts/project/dl_1/imgs/2.webp)

## 데이터 가공

필요한 기능 개발을 위해서 가장 먼저 필요한 것은 게임 내 데이터를 확보 후 가공하는 일이라 판단하였습니다.
게임 소스 파일에 있는 텍스트 데이터 및 인게임에서 대조해가며 필요한 형태로 데이터를 가공했습니다.
![data.ts](/blog/posts/project/dl_1/imgs/3.png)

추후에도 데이터를 불러오거나 가공할 일이 있을 때 편리하도록 타입을 정해두었습니다.
